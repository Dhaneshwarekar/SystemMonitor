# 📊 System Monitor App

A real-time Android system monitoring app built with Android Studio and Java.

## Features
- 🔋 Battery level and charging status
- 💾 Real-time RAM usage
- 📁 Internal storage consumption  
- 📡 Live network download/upload speed
- 📋 Installed apps list with icons
- 📱 Complete device specifications

## Tech Stack
- Android Studio
- Java
- Android SDK APIs (ActivityManager, BatteryManager, StatFs, TrafficStats, PackageManager)
- RecyclerView with custom Adapter
- Material Design Bottom Navigation
- Handler + Runnable for real-time updates
